# src/

Código fuente del proyecto. Organiza los archivos según el lenguaje y framework utilizado:

- **JavaScript/TypeScript**: `components/`, `utils/`, `services/`, `hooks/`
- **Python**: `modules/`, `lib/`, `core/`
- **Rust**: `lib.rs`, `main.rs`, módulos por funcionalidad
- **Go**: paquetes por dominio

Mantén una estructura clara y nombres descriptivos.
